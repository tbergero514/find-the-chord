Quick guide

How to use Find The Chord

1) Pick your key + mode
  • Root Note sets the key center.
  • Scale picks the mode (Major/Minor/Mixolydian/Dorian).
  • Chord Type switches Triads / 7ths / 9ths.
2) Click chords to explore
  • Any chord you click becomes the Current Chord.
  • The Piano highlights its notes.
  • Use Color to quickly switch the current chord between Triad / 7th / 9th.
  • History auto-logs your last 5 clicks.
3) Understand the suggestion grids
  • Diatonic: the 7 chords inside the scale.
  • Smooth Subs: diatonic substitutes sharing 2+ notes.
  • Harmonic Spice: secondary dominants, borrowed chords, tritone subs.
4) Build a progression you control
  • Use Add to append the current chord.
  • Reorder with ← / →, remove with ×.
  • Play plays the whole progression with separation between chords.
  • Use BPM to set playback speed, and Voice-lead to smooth transitions.
  • Copy exports your progression as shareable text.
  • Loop repeats the progression, and Metronome adds a click track.
  • Undo/Redo lets you safely experiment with edits.
Audio tip
If you don’t hear sound, click Enable Audio. Browsers require a user gesture to start audio.
Sharing tip
Your key/mode/progression can be shared via the page URL (it’s encoded in the address bar after you make changes).
